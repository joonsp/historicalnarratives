## Future Updates

Dan Carlin releases new episodes irregularly (typically 1-3 per year). To add future episodes:
1. Check [Podpedia](https://podpedia.miraheze.org/wiki/List_of_Dan_Carlin's_Hardcore_History_episodes) for updates
2. Add new episodes to `src/lib/data/hardcoreHistory.ts`
3. Follow the existing structure with:
   - Episode number
   - Accurate release date
   - Period covered (start/end years)
   - Geographic center coordinates
   - Appropriate zoom level
   - Official episode URL
